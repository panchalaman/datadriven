# Data Driven Strategic Decision Making

Coursework and projects from the Data Driven Strategic Decision Making module.

## Overview

This repository documents the learning journey in data-driven decision making, covering:

- **Business Analytics**: Translating data into actionable insights
- **Predictive Modelling**: Forecasting trends and outcomes
- **A/B Testing**: Experimental design for business decisions
- **Dashboard Development**: Building KPI dashboards
- **Case Studies**: Real-world business scenarios

## Tools & Technologies

- Python, R
- Jupyter Notebooks
- Tableau / Power BI
- SQL
- Pandas, scikit-learn

## Structure

```
├── case-studies/    # Business case analyses
├── notebooks/       # Data analysis notebooks
├── reports/         # Written reports and presentations
└── data/            # Datasets used
```

## Getting Started

```bash
git clone https://github.com/panchalaman/datadriven.git
cd datadriven
pip install -r requirements.txt
```
